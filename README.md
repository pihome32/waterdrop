

# Arduino
## Drop sequence communication
[D123/D1/]
* D : indicator for a sequence string
* 123 : millis from start
* B : Port (B or D)
* 1 : Address
## Pin config
### PORTB
* 0 : Flash3
* 1 : Flash2
* 2 : Flash1
* 3 : Camera focus
* 4 : Camera shutter

### PORTD
* 4 : SOL 4
* 5 : SOL 3
* 6 : SOL 2
* 7 : SOL 1
